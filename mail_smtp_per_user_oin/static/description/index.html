Mail SMTP Server Per User

Mail SMTP Server Per User is a module to allows Send mails from Odoo using your own mail.

Configure SMTP Per User

We can configure different mail servers per user. We only need to specify the owner when creating an outgoing mail server per user. User's email and Username of Outgoing Mail Server should be the same.

Every email will be sent with the reference to a selected owner in an outgoing mail server. Even we also can configure the Incoming Mail Server per user to get a reply back to Odoo.
